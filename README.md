##  Descripción:

Este repositorio reúne prácticas y ejercicios realizados durante la materia de Programación Orientada a Objetos.

El propósito principal es poner en práctica los fundamentos de este paradigma, desarrollando programas bien estructurados, reutilizables y fáciles de mantener a lo largo del tiempo.

A través de las actividades, se representan distintos problemas mediante clases y objetos, simulando elementos del mundo real dentro del código.

* Entender cómo funciona la programación orientada a objetos
* Aplicar conceptos básicos como clases, objetos, atributos y métodos
* Implementar principios clave como:
* Herencia
* Encapsulamiento
* Polimorfismo
* Crear programas más ordenados, claros y escalables

##  Conceptos principales
### Clase:
* Es la estructura base que define las propiedades y comportamientos de un objeto
Objeto:
* Es una instancia de una clase, con datos propios y la capacidad de ejecutar métodos
Encapsulamiento:
* Permite restringir el acceso directo a los datos y controlarlo mediante métodos
Herencia:
* Hace posible que una clase tome características y funciones de otra
Polimorfismo:
* Permite usar métodos con el mismo nombre que se comportan de forma diferente según el contexto
